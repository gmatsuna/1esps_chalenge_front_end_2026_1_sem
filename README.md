# 🏥 CareQuest - Plataforma de Engajamento em Saúde

## 📌 Sobre o Projeto

O **CareQuest** é uma interface web desenvolvida com o objetivo de incentivar hábitos saudáveis por meio de **gamificação**.

A proposta simula uma solução digital para a empresa **CarePlus**, onde usuários podem visualizar seus pontos e recompensas de forma simples, intuitiva e visualmente agradável.

Este projeto foi desenvolvido com foco em **estruturação de layout e design de interface**, sem utilização de lógica de programação.

---

## 🎯 Objetivo

Criar uma interface moderna e responsiva para:

* Visualização de pontos acumulados
* Exibição de recompensas disponíveis
* Navegação entre telas
* Experiência visual agradável e intuitiva

---

## 🚀 Funcionalidades

* 📊 Exibição de pontuação do usuário
* 🎁 Visualização de recompensas
* 🧭 Navegação entre páginas
* 🎨 Interface responsiva
* 🧩 Componentização visual (cards, botões, layout)

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Flexbox
* Responsividade (Media Queries)
* Lineicons / Font Awesome
